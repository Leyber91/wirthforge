# WIRTHFORGE Platform Mockups & Wireframes
*Visual Design for the Consciousness Evolution Platform*

## 🎨 Design Philosophy

### **Energy-First Visual Language**
- **Dark Theme**: Deep space background with neon energy colors
- **Particle Systems**: Every interaction creates flowing particles
- **Energy Flows**: Smooth, organic animations that feel alive
- **Consciousness Colors**: Blue (intelligence), Green (consciousness), Purple (transcendence)

### **Local-First Aesthetics**
- **Minimal UI**: Focus on energy visualization, not interface
- **Smooth Animations**: 60 FPS energy flows
- **Responsive Design**: Works on any device
- **Offline Indicators**: Subtle cues when working locally

## 🎬 Animation Specifications

### **Energy Flow Animations**
```css
/* Lightning Strike Animation */
@keyframes lightning-strike {
  0% { opacity: 0; transform: scale(0); }
  50% { opacity: 1; transform: scale(1.2); }
  100% { opacity: 0; transform: scale(1); }
}

/* Particle Burst Animation */
@keyframes particle-burst {
  0% { transform: translateY(0) scale(0); }
  50% { transform: translateY(-20px) scale(1); }
  100% { transform: translateY(-40px) scale(0); }
}

/* Energy Flow Animation */
@keyframes energy-flow {
  0% { transform: translateX(-100%); }
  100% { transform: translateX(100%); }
}
```
